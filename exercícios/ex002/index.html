Teste de quebra de linha

Explicacoes uteis: se eu quiser que isso < apareça, por ex, ao falar de uma tag, eu preciso usar o código </; seguido do nome da tag. NÃO TEM BARRA, foi só pra dar o exemplo. São esses caracteres aqui juntos: "& l t ;", incluindo o ponto e virgula. esse daí, lt, é pra isso < . Abertura de tag. Significa "menor que (less than)", que é o nome desse símbolo.
Para mostrar o símbolo de fechamento de uma tag você usa esse código aqui " & g t ;" sem aspas e tudo junto, incluindo o ponto e vírgula. Significa "maior que (greater than)" que é o nome do símbolo.
Para fazer comentários no seu código, use isso aqui que está entre aspas tudo junto: " < ! - - - -". 4 hífens, escrevendo no meio deles.

Hora de aprender a colocar símbolos especiais no código.
