Semântica na HTML5 - formatação

A HTML5 mudou o conceito do html, fazendo-o focar em semântica. Assim, algumas tags foram trocadas, porque se referiam especificamente à forma. Agora, precisamos olhar as novas tags e saber como elas são feitas.

Negrito/destaque

Antigamente, se usava de forma não semântica. Você já sabe qual é. Ainda funciona, mas não é semântica, não quer dizer nada. Atualmente, você coloca STRONG. Significa que você quer dar destaque no seu texto, ou seja, qual o motivo dessa mudança.

Itálico/ênfase

Novamente: itálico era só a forma. agora, usamos EM - emphasis. Ou seja, damos ênfase ao texto.

Além disso, quando um leitor de tela vai ler o texto, comandos semânticos fazem realmente dar ênfase/destaque ao seu texto.

Texto Marcado (marcador de texto)

Podemos criar um texto marcado usando a tag mark. Para mudar a cor, só com CSS. Isso aí foi css. Vamo fazer outro texto marcado aqui.

A gente pode mudar todos os marks ao mesmo tempo, também. Para isso, a gente deixa só mark mesmo, e na head tu faz uma tag style e cria um seletor mark: background color. Assim quando tu atualizar vai estar para todo mundo.

Texto pequeno

Tiraram a tag de texto grande, mas ainda existe a tag small para texto pequeno. Mas pense no contexto ao usar, caso não haja modifique em css.

Texto deletado

Marcamos um texto como excluido usando del. Para inserir (ou sublinhar, mas com sentido), use ins.

Texto sobrescrito e subescrito.

Para inserir coisas tipo x ao quadrado: x20 (coloca o 2 normal): sobrescrito, tag sup. Para inserir coisas tipo h2o (coloca o 2 normal): subescrito, tag sub.
