Início

Cria uma pasta dentro da sua e coloca um arquivo lá, é ele que vamos fazer download. No caso, a pasta livro. Se for algo muito grande, vale compactar seu arquivo num zip.

Links para download

Para fazer isso você vai colocar um link normal com o arquivo que você quer baixar, mas adiciona o atributo download com o valor configurado para o nome do arquivo (repete o nome do arquivo mesmo) e atributo type, que vai dizer o que tá sendo baixado. Ou seja, vai ficar um a href="CAMINHO" download="NOME DO LIVRO" type="tipo do arquivo".

Mas como que eu sei o type do meu arquivo?

Lista de media types: lista oficial da IANA.org !! Mas além disso, aqui temos umas opções de media types usados comumente:

  • application/zip (arquivos zip)
  • application/pdf
  • text/html
  • text/css
  • text/javascript
  • video/mp4
  • video/H264
  • video/JPEG
  • audio/aac
  • audio/mpeg
  • font/ttf
  • image/jpeg
  • image/png
